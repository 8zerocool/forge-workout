Plan Improvements

Biggest upgrades

These changes make the plan more aligned with an aesthetic, athletic, long-term physique.

🦾

More side delts

Add lateral raises 2–3 times per week. Side delts create the wide shoulder look that dominates aesthetics.

🧱

More upper chest

Incline press plus incline fly or incline machine work creates the squared upper torso that fills a shirt.

🏹

Add rows

Rows balance pressing, improve posture, and help build a thicker athletic back with visible definition.

📈

Progression system

Use double progression: hit the top of the rep range, then increase weight and repeat. Track your PRs.

⚠️ Recovery rules

Do not take every lifting set to failure.
Most sets should stop with 1–2 reps in reserve (RIR).
Keep Sunday lighter so Monday legs stay strong.
Deload every 5–6 weeks if joints, sleep, or performance decline.
Fuel kickboxing days with enough carbs and hydration.
